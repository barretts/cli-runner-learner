cli-runner-learner

Autonomous CLI tool learning & skill export

PTY probing. Behavioral profiles. Adapter presets. Zero hardcoding.

View on GitHub Quick Start

How It Works

1

Discover

Run the tool with --help and parse its capabilities: subcommands, flags, interactivity, and reduce-motion env vars.

2

Probe

Launch the tool in a real PTY, apply adaptive probe strategies, and record every byte of terminal I/O as JSONL transcripts.

3

Learn

Classify output into a 7-state machine (startup, ready, working, thinking, prompting, completed, error) and extract glob patterns.

4

Export

Generate driving skills (markdown) and adapter presets (TypeScript/JSON) ready to drop into AgentThreader.

What Gets Generated

AdapterPreset TypeScript / JSON
Skill Markdown Per-tool driving guide
ToolProfile Behavioral state machine
Compiled Skills 5 IDE targets

CLI in Action

$ clr learn --tool crush --command crush --heal auto
[discover] Parsing --help output...
[discover] Found 12 subcommands, 9 flags
[probe:1] Strategy: observe (30s)
[probe:1] Classified 6 segments, extracted 14 patterns
[probe:2] Strategy: multi_turn (45s)
[probe:2] Classified 8 segments, extracted 19 patterns
[learn] Profile confidence: 81.9% -- converged
[learn] Saved: profiles/crush.json

$ clr export-adapter --tool crush
[export-adapter] Generated: generated/adapters/crush-preset.ts
[export-adapter] Profile confidence: 81.9%
[export-adapter] Overrides: applied

$ clr export-skill --tool crush
[export-skill] Generated: generated/skills/crush.md
[export-skill] States: startup, ready, working, prompting, thinking, completed, error

Quick Start

$ npm install -g cli-runner-learner
added 3 packages in 4s

$ clr learn --tool my-tool --command my-tool
[learn] Learning my-tool behavior...

$ clr export-adapter --tool my-tool
[export-adapter] Generated: generated/adapters/my-tool-preset.ts

$ clr export-skill --tool my-tool
[export-skill] Generated: generated/skills/my-tool.md

Requires Node.js ≥ 20. The clr CLI is the main entry point for all commands.
